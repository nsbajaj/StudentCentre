Help information

  • Available Courses: View list of all courses available.
  • Students: View list of all courses available. Once logged in, a professor/advisor can select a student and view their details such as credit history, courses selected for next term.
  • Cart: The cart allows students to view and select courses for their next term. In order to view the cart, a student must first view their details.
  • Create: The purpose of this component is to enable a NEW user account to be created.
  • Activate: The purpose of this component is to enable a user account to be activated.
  • Login: The purpose of this component is to enable a user to login.
  • Logout: The purpose of this component is to enable a user to logout.
